# Выполнение задания: публикация 1С на веб-сервере Apache

## Что сделано

1. **Проверена разрядность платформы 1С** — учебная версия 32-битная.

2. **Установлен модуль расширения веб-сервера** в платформе 1С через изменение установки.

3. **Скачан и установлен Apache 2.4** (32-битная версия) в папку `C:\Apache24`.

4. **Apache установлен как служба Windows** и запущен.

5. **Выполнена публикация базы «УправлениеИТФирмой»** через Конфигуратор 1С:
   - Имя публикации: `IT-company-management`
   - Каталог: `C:\Apache24\htdocs\IT-company-management`

6. **База успешно открыта в веб-клиенте** по адресу `http://localhost/IT-company-management`.
